
1. 显示当前数据库
```
SHOW DATABASES
```

2. 增添数据库
```
CREATE DATABASE database_name
```
3. 删除数据库
```
DROP DATABASE database_name
```
4. 使用数据库
```
USE test_db
Database changed #数据库已更改
```
5. 建立表
```
create table 表名称
```
6. 显示数据库里面的所有的表
```
show tables
```
7. 显示某一个表的结构
```
desc T_student #显示名为T_student的表
```
8. 创建表
```
CREATE TABLE Shohin
(shohin_id CHAR(4) NOT NULL,
shohin_mei VARCHAR(100) NOT MULL,
shohin_burui VARCHAR(32) NOT NULL,)
;
```
